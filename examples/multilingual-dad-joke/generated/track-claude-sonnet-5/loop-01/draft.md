# Loop 01 Draft — "Hi Hungry, I'm Dad"

## Mechanism
A child states a feeling using an "I am [state]" construction. Dad deliberately
mishears the feeling-word as if it were the child's given name, and responds
with the standard first-introduction greeting, introducing himself as "Dad."
The comic engine is category confusion (mistaking a description for an
identity) — a universal, non-phonetic mechanism that should be rebuildable in
any language using that language's own self-introduction formula.

## English
> Dad, I'm hungry.
> Hi Hungry, I'm Dad.

## Français (French)
> Papa, j'ai faim.
> Salut Faim, moi c'est Papa.

## Español (Spanish)
> Papá, tengo hambre.
> Hola Hambre, soy Papá.

## 日本語 (Japanese)
> パパ、お腹すいたよ。
> はじめましてお腹すいた、パパだよ。

## Notes
"Hungry" was chosen because it is the most iconic version of this joke format
in English. It has not yet been checked for grammatical fit against French,
Spanish, and Japanese equivalents — see self-critique.md.
