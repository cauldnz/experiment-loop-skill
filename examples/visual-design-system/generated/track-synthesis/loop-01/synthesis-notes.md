# Synthesis 01 — evidence-led combination

## Parent contributions accepted

- **Editorial-02 hierarchy and grid:** accepted. Opus scored its primary hierarchy 5/5 and called it the cleanest of four; Terra also scored hierarchy, clarity, coherence, and polish 5/5. The synthesis keeps a dominant serif title, explicit DATE/TIME/VENUE labels, rule-led alignment, and an isolated CTA.
- **Editorial-02 token discipline and accessibility:** accepted. Its repaired muted token measured 5.28:1 and its two real variants proved reuse. This loop extends that model to color roles, typography, spacing, radii, layouts, component states, constraints, and three event variants.
- **Generative-02 modular loop/signal language:** accepted. Every panel agreed it was the strongest source of distinctiveness; Gemini ranked it first overall and Opus called it the strongest image. The routing module remains a large, dedicated brand surface.
- **Generative-02 text/graphic zoning:** accepted in principle, strengthened in measurement. An 80px structural gutter and 68.738px rendered title-to-panel separation replace the parent's under-measured hemisphere boundary.

## Parent contributions rejected or repaired

- **Blue CTA (`#3B82F6`) with white text:** rejected. Opus independently recomputed 3.68:1, below normal-text AA. The new `#173B57` surface measures 11.67:1 with white.
- **Generative title/panel boundary:** rejected as unsafe. Opus found about 24px intrusion. The rendered synthesis title ends at x=691.262; the module starts at x=760, leaving 68.738px and clearing the declared 64px minimum.
- **Empty metadata icon chips:** rejected. Explicit labels convey semantics directly.
- **Generative palette-only tokens and uncritical “Perfect” self-review:** rejected. Parent evidence showed no spacing, geometry, component contract, or variant mechanism, and contradicted the self-score.
- **Editorial full-width composition without a bespoke digital brand object:** rejected. It was the safest system but Gemini reasonably dissented that it bordered on generic print mimicry.
- **Editorial fixed 190px badge:** not carried forward unchanged. Loop 01 publishes smaller explicit character budgets, but still uses fixed display/compact widths; this remains a testable weakness rather than being hidden.

## Preserved panel dissent

- **Gemini:** preferred generative-02 because its instrument-panel aesthetic was more memorable and digitally native.
- **Opus and Terra:** preferred editorial-02 on the whole brief because hierarchy, information clarity, objective measurements, and real variant tokens were stronger.
- **Synthesis position:** neither view is flattened. Editorial structure governs reading order and system behavior; generative language governs brand recognition. This is not an independent promotion verdict.

## Loop strengths

- Visual hierarchy is title → explicit metadata → CTA; the signal module is supporting but distinctive.
- Both independently detected generative defects are objectively repaired.
- Three compact event variants demonstrate shared components, responsive stacking, accent states, and routing patterns.
- SVGs are standalone, use system font fallbacks only, and include accessible names/descriptions.

## Remaining weakness observed

The first real Inkscape render exposed a concrete compact-layout failure that the analytical estimate missed: hybrid venue `HUB A + WEB` renders to x=779.358 while its card ends at x=776. It visibly escapes the component by 3.358px (and the inner frame by 15.358px), so Loop 01's no-clipping/layout-containment gate fails. The published 11-character venue budget is therefore disproven by the artifact itself. Fixed-width badges and the two-line display-title assumption remain related capacity risks.

## Next hypothesis

A width-aware compact metadata component, a compact-density display title tier, content-driven badge widths, and a deliberately long-content event can repair the measured overset and improve resilience without weakening the title-first hierarchy or the measured 64px minimum title/module separation.
