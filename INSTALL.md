# Install the Experiment Loop Skill

## Windows

From this repository root:

```powershell
.\scripts\install.ps1
```

To overwrite an existing copy:

```powershell
.\scripts\install.ps1 -Force
```

## macOS/Linux

From this repository root:

```bash
bash scripts/install.sh
```

To overwrite an existing copy:

```bash
bash scripts/install.sh --force
```

## Manual install

Copy this repository folder to:

```text
~/.copilot/skills/experiment-loop
```

After install, start a new Copilot session or reload skills if your environment supports skill reload.
